# SOC Labs

This repository contains hands-on SOC (Security Operations Center) investigations focused on real-world incident analysis. Each case is built and investigated using my own [SOC home lab](https://github.com/4belSec/soc-home-lab) — real attacks executed against my own infrastructure, detected and analyzed using Splunk.

## Objective

To demonstrate practical SOC analyst skills, including:

- Log analysis
- Threat detection
- Incident investigation
- Detection engineering (SPL / Splunk)

## Cases

- [RDP Brute Force Attack Analysis](cases/01-rdp-brute-force.md) — a real brute force attack executed via Hydra against an RDP endpoint, detected and correlated in Splunk using custom SPL queries.

## Goal

Build a portfolio that reflects real, hands-on SOC analyst capabilities — grounded in actual attack execution and log analysis, not theoretical exercises.
